WooCommerce plugin with Payment for BANKNET in Viet Nam only.

== Description ==
Requirement: Registered with BANKNET, NAPAS first to get your Merchant Id, Access Code, Secure Hash Secret for using this plugin.

This plugin enables BANKNET, NAPAS payment gateway support for your WooCommerce sites. It Allows you to use BANKNET, NAPAS Gateway to pay via more than 20 Viet Nam Banks with the WooCommerce plugin. 

It uses the redirect method, the user is redirected to BANKNET, NAPAS payment gateway page so that you don't have to install an SSL certificate on your site.

== Installation ==
0. Registered with BANKNET, NAPAS first to get your Merchant Id, Access Code, Secure Hash Secret for using this plugin.
1. Ensure you have latest version of WooCommerce plugin installed ( 2.2 or above )
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit the WooCommerce settings page, and click on the Checkout tab.
5. Select the BANKNET Payments and click Save changes. 
6. Use the settings button to configure BANKNET payment gateway.
6. On your BANKNET setting page, check the Enable BANKNET Payment Module check box.
7. Add your Merchant Id, Access Code, Secure Hash Secret and other setting.
8. Click Save changes.
